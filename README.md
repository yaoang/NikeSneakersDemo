# Nike Sneakers Shop Demo
> Author Ang Yao

## Demo of Nike sneakers Shop

### How to install libraries?
```
npm run install-all
```
### How to build project?
```
yarn run build-fe
```

### How to run test?
```
yarn test
```

### How to test front-end?
```
yarn test-fe
```
and Press `a`

### How to start the website?
```
yarn start
```
Then you can open [http://localhost:3000](http://localhost:3000)
