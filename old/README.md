# Nike Sneakers Shop Demo
> Author: Ang Yao

## Demo of Nike sneakers Shop

### Directory and Files
Server Entrance
```
./server.js
```

Front End Directory
```
./static
```

Test Files
```
[FileName].test.js
```

### How to install libraries?
```
npm run install-all
```
### How to build project?
```
yarn run build-fe
```

### How to run test?
```
yarn test
```

### How to test front-end?
```
yarn test-fe
```
and Press `a`

### How to start the website?
```
yarn start
```
Then you can open [http://localhost:3000](http://localhost:3000)
